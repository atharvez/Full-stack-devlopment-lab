# Assignment 4: SkyInsight Weather Intelligence

## ⛅ Overview
SkyInsight is a high-fidelity weather visualization dashboard I built using **Vanilla JavaScript** and **Chart.js**. This project represents a deep dive into data manipulation, focusing on transforming complex asynchronous API responses into interactive, readable intelligence.

---

## 🚀 Experience Highlights

### 📊 Data Visualization
- **Dynamic 24h Trends**: High-performance temperature charts powered by Chart.js.
- **Visual Distribution**: Condition mix breakdown (clear vs cloudy vs rain) using donut charts for regional analysis.
- **Metric Toggling**: Integrated persistence for Celsius vs Fahrenheit preferences via `localStorage`.

### 🌍 Global Connectivity
- **City Intelligence**: Real-time integration with the **Open-Meteo Geocoding API** for worldwide location discovery.
- **Live Forecasts**: Fetches real-time atmospheric data with a zero-delay, zero-key infrastructure.

### 💎 Design Philosophy
A "Midnight Premium" UI featuring:
- **Glassmorphic Glass Panels** for atmospheric depth.
- **Lucide Icon Integration** for a sharp, recognizable interface.
- **Custom CSS Micro-animations** to enhance user engagement.

---

## 🛠️ Technical Stack
- **Standard**: HTML5, CSS3, JavaScript (ES6+)
- **Library**: Chart.js v4+
- **Icons**: Lucide Assets
- **Engine**: Open-Meteo REST API
