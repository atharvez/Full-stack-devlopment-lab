---

## 🌍 GitHub Pages Setup

To host your project:

1. Go to **Repository Settings → Pages**
2. Select:

   * Branch: `main`
   * Folder: `/root`
3. Save and wait for deployment

Your site will be live at:

```
https://atharvez.github.io/Full-stack-devlopment-lab/
```
